# StudioAssistant

A way to keep track of a band, members, and contact information for quick lookup, im sure an address book would be fun too but I had to make something with winforms

It saves the information in Json files, Each Artist has Contact Info, Date to set for Last Contact and Enough to start keeping track of the artists
I would like to add a theme chooser, and some more basic functionality, It does not delete Band Members, but you can write over them.
Basic Information on the main Page and a few details in the other.  Just a handy thing for the studio, I would like it to update my projectFiles so it could link that
and have the project files also get saved with a date of last worked on.  Could also have markdowns for the analog gear, and more notes for sessions.
